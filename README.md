## 조작키

* 방향키 &nbsp;&nbsp; &nbsp;↑ &nbsp;&nbsp; : 회전
* 방향키 ← &nbsp; → : 좌우 이동

<br/>

* SPACE : Hard drop
* P &nbsp; &nbsp; &nbsp; &nbsp; : 일시 정지
* ESC &nbsp;&nbsp;&nbsp; : 종료
